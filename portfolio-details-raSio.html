Contexte

L'application raSio est un projet JAVA, ou nous utilisons des requêtes préparées afin de bien sécuriser l'application, l'objectif de cette application est de lier nos connaissances en IHM (interface homme-machine), en programmation orientée objet mais aussi en base de données.

Les Besoins

Rasio est une application Java permettant à un utilisateur d'organiser une course automobile, après s'être authentifié, l'utilisateur peut effectuer différentes actions. Il à la possibilité de créer différents éléments, il peut ajouter une écurie, une course, un circuit, et du personnel. Il a également la possibilité d'afficher tous ces éléments dans un tableau. Il peut également les supprimer ou les rechercher. L'application permet également d'afficher ces éléments sous différents formats d'affichages, comme le CSV, le XML, et le JSON.
